##🛒 Fresh Mart — Online Grocery Store

A full-stack web application developed as a group project that provides a complete online grocery shopping experience. Fresh Mart allows customers to browse and search for fresh vegetables, fruits, and meats, while administrators can efficiently manage product inventory through a dedicated dashboard.

---

## 📌 Overview

Fresh Mart is a responsive grocery shopping platform designed to simplify online grocery management and purchasing. Customers can explore products by category, search for specific items, manage personal profiles, and view featured products.

The application also includes an administrative panel that provides complete control over inventory management, allowing administrators to add, update, and remove products from the system.

The system is powered by a PHP backend with MySQL database integration and runs locally using Apache through XAMPP.
